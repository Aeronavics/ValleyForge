====== Modular Code Uploading Tool: Arduino STK500v2 Serial Protocol Module ======
This module can be used to program the ArduPilot Mega boards using the arduino serial bootloader on them.

===== Module Parameters =====
This module uses a number of parameters given to the uploader with the -C command line switch.
