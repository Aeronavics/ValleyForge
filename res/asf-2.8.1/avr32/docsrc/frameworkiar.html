Introduction

Let's take the example of the GPIO driver module.

  • EVK1100 : /drivers/gpio/peripheral_bus_example/at32uc3a0512_evk100/iar
  • EVK1105 : /drivers/gpio/peripheral_bus_example/at32uc3A0512_evk1105/iar
  • EVK1101 : /drivers/gpio/peripheral_bus_example/at32uc3B0256_evk1101/iar
  • EVK1104 : /drivers/gpio/peripheral_bus_example/at32uc3A3256_evk1104/iar
  • STK600-RCUC3L0 : /drivers/gpio/peripheral_bus_example/at32uc3l064_stk600-rcuc3l0/iar
  • UC3C_EK : /drivers/gpio/peripheral_bus_example/at32uc3c0512c_revc_uc3c_ek/iar
    • The /IAR folder contains a full IAR project with:

  • an IAR EWAVR32 workspace file(gpio_peripheral_bus_example.eww: double-click on this file to open the whole project.),
  • an IAR EWAVR32 project file(gpio_peripheral_bus_example.ewp),
  • an IAR EWAVR32 debug configuration file(gpio_peripheral_bus_example.ewd).

    AVR UC3L064

    AVR UC3C0512C
